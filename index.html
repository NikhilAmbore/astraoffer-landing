After download

Setup guide — 5 minutes.

Follow these steps exactly. Takes one time only.

1
Open the DMG and drag to Applications
Double-click the downloaded .dmg file. Drag "AstraOffer Assistant" to your Applications folder. Eject the DMG.
2
Bypass Gatekeeper — right-click → Open
macOS will say "damaged and can't be opened" if you double-click. Instead: right-click the app → click Open → click Open in the dialog. You only do this once.
⚠ Alternatively, run in Terminal:
xattr -cr "/Applications/AstraOffer Assistant.app"
3
Grant Screen Recording permission
Go to System Settings → Privacy & Security → Screen Recording → enable AstraOffer Assistant. This is required for screen capture and audio loopback. Restart the app after granting.
4
Get your Claude API key (free)
Go to console.anthropic.com → API Keys → Create Key. Copy it. Paste it into the AstraOffer Assistant API Keys tab. The free tier gives you $5 of credit — enough for ~500 interview answers.
5
Get your Groq API key (free, no credit card)
Go to console.groq.com → API Keys → Create. Copy and paste into the app. Groq runs Whisper transcription — completely free for personal use.
6
Upload your resume and start your session
In the app, go to Setup → paste your resume and job description → click Start Session. During the interview press Cmd+Shift+S to start listening or Cmd+Shift+C to capture a coding problem. That's it.

Yes. The app uses setContentProtection(true) which calls macOS's CGWindowSharingNone at the OS level. This excludes the window from the CGWindowListCreateImage API that Zoom, Teams, Meet, OBS, and all other screen recording software use to capture your screen. The window pixels are never sent to the GPU framebuffer that screen capture reads from. It is not just "hidden" — it is architecturally excluded from capture.
