After download

Setup guide — 5 minutes.

Follow these steps exactly. Takes one time only.

1
Install the app
Mac: Double-click the .dmg, drag "AstraOffer Assistant" to Applications, eject the DMG.
Windows: Run the .exe installer, follow the prompts, launch from the Start menu or Desktop shortcut.
2
First launch — bypass the OS security warning
Mac: macOS will say "damaged and can't be opened." Right-click the app → Open → Open. Or run:
xattr -cr "/Applications/AstraOffer Assistant.app"
⚠ This only needs to happen once.

Windows: Windows Defender may warn "Unrecognized app." Click "More info" → "Run anyway." This is because the app is unsigned.
3
Grant Screen Recording permission (Mac only)
System Settings → Privacy & Security → Screen Recording → enable AstraOffer Assistant. Restart the app after granting.
4
Get your Claude API key (free)
Go to console.anthropic.com → API Keys → Create Key. Copy it. Paste it into the AstraOffer Assistant API Keys tab. Free tier gives $5 credit — enough for ~500 interview answers.
5
Get your Groq API key (free, no credit card)
Go to console.groq.com → API Keys → Create. Copy and paste into the app. Groq handles real-time audio transcription — completely free for personal use.
6
Upload your resume and start your session
In the app → Setup → paste your resume and job description → Start Session. During interviews: Cmd+Shift+S (Mac) / Ctrl+Shift+S (Windows) to listen, or Cmd+Shift+C / Ctrl+Shift+C to capture a coding problem.
