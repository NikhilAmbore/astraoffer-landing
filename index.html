Interviewer question
"Design a notification service that can handle 10 million users."
AstraOffer answer
I'd start with requirements: push, email, SMS. At 10M users, fan-out is the core problem. I'd use a message queue — Kafka — with per-channel consumer groups. Notification preferences sit in Redis for sub-millisecond reads. For email/SMS, delegate to SendGrid/Twilio via async workers. Rate-limit per user at the queue layer. Store delivery receipts in Cassandra for scale. At 10M that's ~50K events/sec at peak — fully horizontal.

Say while coding: "For each number I check if its complement is already in the map…"
Complexity: O(n) time · O(n) space

After download

Setup guide — 5 minutes.

Follow these steps exactly. Takes one time only.

1
Install the app
Mac: Double-click the .dmg, drag "AstraOffer Assistant" to Applications, eject the DMG.
Windows: Run the .exe installer, follow the prompts, launch from the Start menu or Desktop shortcut.
2
First launch — bypass the Mac security warning
Mac — if you see "damaged and can't be opened":
Open Terminal (⌘ Space → Terminal) and run:
xattr -cr "/Applications/AstraOffer Assistant.app"
Then launch the app normally. ⚠ One-time only — the app opens instantly from then on.

Windows — if you see "Windows protected your PC":
Click More info → Run anyway. Safe to proceed.
3
Grant Screen Recording permission (Mac only)
System Settings → Privacy & Security → Screen Recording → enable AstraOffer Assistant. Restart the app after granting.
4
Sign in with your AstraOffer account
Open the app → click Sign In → log in with your AstraOffer account. If you don't have one yet, create a free account at astraoffer.org in 30 seconds. No credit card needed for the free plan. AI is included — no extra keys required.
5
Upload your resume and start your session
In the app → Setup → paste your resume and job description → Start Session. During interviews: Cmd+Shift+S (Mac) / Ctrl+Shift+S (Windows) to listen, or Cmd+Shift+C / Ctrl+Shift+C to capture a coding problem.
